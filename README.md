4Dancer
==================

목적
-----------------------------
연습실 공유할 사람을 찾기!

주요기능
-----------
- 글을 올려 연습실을 공유할 사람 찾기
- 내가 쓴 글 확인 가능
- 내가 연습하고 싶은 곳의 주소를 검색하면 그 주변의 연습실 글 찾기 기능
- 원하는 연습실 공유 글에 댓글 달기 기능

사이트 이용 방법
---------------------------
- 회원가입 후 로그인
- 업로드 글 페이지로 접속, 제목과 본문, 연습실 위치, 최대 인원수, 날짜를 입력 
- 원하는 연습실 공유 글에 댓글을 달며, 사람 찾기.
